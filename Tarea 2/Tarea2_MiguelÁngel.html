3. Planteamiento¶

3.1. El modelo de regresión¶

Dado un conjunto de datos $\{y_{i}, x_{i1},\dots,x_{in}\}_{i=1}^{N}$, el modelo de regresi√≥n asume que existe una relaci√≥n lineal entre la variable dependiente $y$ con las variables independientes $x$. Esta relaci√≥n se modela por medio de un t√©rmino de error $\varepsilon_{i}$, el cual es una variable aletoria que agrega un par√°metro de ruido a dicha relaci√≥n. Por lo cual, de manera general, el modelo puede tomar la siguiente forma [Linear Regression - Wikipedia]:

$$y_{i}=\beta_0+\beta_0x_{i1}+\dots+\beta_{n} x_{in}+\varepsilon_{i}\quad\text{, }i=1,\dots,n$$

O de manera matricial: $$\mathbf{Y}=\mathbf{X}\mathbf{\beta}+\mathbf{\varepsilon}$$

Por lo cual, el objetivo del modelo lineal es encontrar el valor de los par√°metros $\beta$ y describir la relaci√≥n entre las dos variables.

3.2. Estimación de los parámetros¶

Para la estimaci√≥n de los par√°metros, se puede utilizar el m√©todo de minimos cuadrados. En este caso, el m√©todo lo que busca es minimizar la siguiente ecuaci√≥n para el modelo lineal simple (Hocking, 2013):

$$Q(\beta_{0},\beta_{1})=\sum_{i=1}^{N}(y_{i}-\beta_{0}-\beta_{1}x_{i})^{2}$$

Dicha minimizaci√≥n se lleva acabo realizando las derivadas parciales de la funci√≥n $Q(\beta_{0},\beta_{1})$ e igualando a 0, de tal manera que:

$$\frac{\partial Q}{\partial \beta_{0}}=0\quad\text{y}\quad\frac{\partial Q}{\partial \beta_{1}}=0$$

De esta manera, se obtiene que los par√°metros ser√°n igual a:

$$\hat{\beta}_{0}= \frac{1}{N}\sum_{i=1}^{N}(y_{i}-\beta_{1}x_{i})\\ \hat{\beta}_{1}= \frac{S_{xy}}{S_{xx}}$$

Dichos resultados se pueden generalizar a N variables, no obstante, la metodolog√≠a es an√°loga. As√≠, finalmente, se obtiene que la regresi√≥n lineal estar√° dada por la siguiente ecuaci√≥n:

$$\hat{y}=\hat{\beta}_{0}+\hat{\beta}_{1}x$$

A modo de ejemplo, la siguiente gr√°fica presenta un una regresi√≥n lineal para un conjunto de datos. En este caso, cuando se realiza una regresi√≥n con una sola variable independiente, se le llama regresi√≥n lineal simple.
